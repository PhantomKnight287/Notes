### Ways  We Can Styles Components.
1. By Writing Css in `_.component.ts`
2. By defining a `stylesUrl` property pointing to a css file in the directory in Component's Decorator
3. `style` tag in Component's HTML.
4. `link` tag in Component's Html or in `index.html` file present in project's root.
