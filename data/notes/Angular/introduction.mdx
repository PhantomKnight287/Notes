[Angular](https://angular.io) is an Open Source UI framework which is used to build apps for Web, Mobile Web, Native Mobile and Native Desktop.

Angular is Developed by Google and uses TypeScript by default.

# Why Angular
1. Supports Various Platforms.
2. It is Maintained By Google.
3. Better Tooling.
4. Easy Routing.
5. Markup is Clean and simple.
6. Applications are easy to test.
7. Uses TypeScript
